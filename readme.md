# Estrutura React Native
Esse projeto é uma referência para iniciar aplicativos com [React Native](https://reactnative.dev/]).

Para facilitar o entendimento foi usado como builder o [Parcel](https://parceljs.org/).

## Como iniciar
Projetos React Native usam o [Expo go](https://expo.dev/go) para facilitar a visualização do aplicativo pois facilmente o app é transferido para o próprio smartphone realizando a leitura do QrCode gerado no terminal.

Depois de realizar o download do projeto e extrair em um local de sua preferência você deve abrir o terminal **CMD no Windows** ou **Terminal no Linux** os comandos abaixo:

Faz a instalação das dependências do projeto que estão descritas no arquivo **package.json**
```
npm i
```

Executar o projeto, abre um qrcode no terminal que deve ser lido com aplicativo Expo
```
npm start
```
